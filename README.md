* What is a web page?

    * CSS - how to style the content
    * JavaScript - code to make the page interactive

        * head
        * title
        * body
        * headers
        * p
        * strong and em
        * section
        * ul, ol, and li
        * article
        * div
    * Tag attributes
        * a
        * img
        * link
        * span
        * The class attribute
* CSS
    * Selecting a tag
    * Selecting a class
    * syntax
    * Units: px, %, em, rem
    * CSS properties
        * Text and fonts
            * color

            * font-size
            * font-family
            * font-weight
            * text-decoration
            * text-align
        * Blocks
            * background-color
            * padding
            * margin
            * border-color
            * border-width
            * border-style
            * border-radius
            * width
